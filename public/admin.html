Weather

Sign up at openweathermap.org (free account), then subscribe to One Call API 3.0 under "API keys." Free tier includes 1,000 calls/day.

Calendar

  1. Go to Google Cloud Console and create a new project
  2. Enable the Google Calendar API
  3. Go to OAuth consent screen — choose "External," fill in the app name, and add your email
  4. Under "Scopes," add calendar.readonly
  5. Under "Test users," add your Gmail address
  6. Go to Credentials → Create Credentials → OAuth client ID → Web application
  7. Add http://localhost:3333/auth/google/callback as an authorized redirect URI (replace with your Pi's address if accessing remotely)
  8. Copy the Client ID and Client Secret below
From a Mac: Open Calendar → right-click a calendar → Share Calendar → check "Public Calendar" → copy the URL.

From iCloud.com: Open Calendar → click the share icon next to a calendar → check "Public Calendar" → copy the link.

The URL starts with webcal:// — paste it as-is. Add one entry per calendar you want to show.

Pollen

Uses the same Google Cloud project as Calendar. Enable the Pollen API, then go to Credentials → Create Credentials → API key.

Location

Enter your coordinates for weather and pollen data. You can find them by searching your address on latlong.net.
